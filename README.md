# Algorithm-Study

### TL;DR
```
일주일에 본인 역량에 맞는 문제 5개를 선정 
```

<br/>

### 코드 정리에 들어갈 내용
* 해당 문제 출처(백준, 프로그래머스 등)
* 어떤 유형의 문제인가?
* 어떤 식으로 문제를 접근하고 해결하고자 했는가?
* 그 방법을 통한 문제 해결 사고의 흐름
* 부족하다 생각되는 것, 애매한 것, 혹은 좋았던 점 등의 Discussion point는 Issue를 통해 질문하고 소통하는 창구로 만들고자 한다.
